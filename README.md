# Travel-Way - Travel & Tourism Booking Website

A full-stack responsive web application for browsing travel destinations and booking packages online. Built for **Web Technology (BIT233)** assignment at Texas College of Management & IT.

## Features
- User Registration & Login (with password hashing)
- Browse Destinations (Cities, Treks, Packages)
- Book Travel Packages (with date & people selection)
- View Booking History (Dashboard)
- Responsive Design (Mobile + Desktop friendly)
- Admin CRUD (Add/Edit/Delete Destinations – protected route)
- Form Validation (Client-side JS + Server-side Flask-WTF)
- Database Integration (SQLite + Flask-SQLAlchemy)
- Google OAuth Login (optional setup)

## Tech Stack
**Frontend**
- HTML5, CSS3, Bootstrap 5
- JavaScript (form validation)

**Backend**
- Python Flask
- Jinja2 Templating
- Flask-SQLAlchemy (ORM)
- Flask-Login (Authentication)
- Flask-WTF (Forms)

**Database**
- SQLite (development)

**Tools**
- VS Code
- Git & GitHub
- Python 3.10+
